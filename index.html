Implementation
Implementation

Initialization

  1. Set up connections for all sensors and modules.
  2. Initialize serial communication and the LCD display.
  3. Configure the WiFi module (ESP8266-01) for internet connectivity.
Implementation

Read Sensor Data

  1. Read data from the TMP102 temperature sensor, soil moisture sensor, and water flow sensor.
  2. Display sensor data on the LCD screen.
Implementation

Process Sensor Data

  1. Compare soil moisture reading against a predefined threshold.
  2. Prepare to activate the water pump if the soil moisture level is below the threshold.
Implementation

Control Water Pump

  1. Turn on the water pump using a MOSFET if soil moisture is low.
  2. Monitor the water flow sensor during irrigation.
  3. Turn off the pump once soil moisture reaches an adequate level.
Implementation

Send Data to Mobile

  1. Establish communication with the mobile app via the WiFi module.
  2. Send sensor data to the mobile app.
  3. Receive and execute control commands from the app.
Implementation

Manual Control via Mobile

  1. Check and execute commands from the mobile for manual operation.
Implementation

GSM Module Communication

  1. Set up the GSM module (SIM800L).
  2. Send alerts and updates to the user's phone via SMS if necessary.
Implementation

Stepper Motor Control

  1. Initialize the stepper motor driver (EasyDriver).
  2. Execute predefined stepper motor movements based on commands.
